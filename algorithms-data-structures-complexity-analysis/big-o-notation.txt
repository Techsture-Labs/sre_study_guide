- A relative representation of the complexity of an algorithm.
  - Constant
    - O(1)
    - Picking a list element directly
  - Linear
    - O(n)
    - Looking for a value by iterating through a list
  - Logarithmic
    - O(log n)
    - Binary Search Tree (Bisection Search)
  - Log Linear
    - O(n log n)
    - Merge Sort
    - Heap Sort
    - Quick Sort
  - Quadratic
    - O(n^2)
    - Bubble Sort
    - Insertion Sort
    - Selection Sort
    - Traversing a two-dimensional array
  - Cubic
    - O(n^3)
  - Exponential
    - O(2^n)
